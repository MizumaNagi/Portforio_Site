美女と野獣

Details

作品名
美女と野獣
ジャンル
2Dアクション
ゲーム概要
歪んだ童話を舞台に、美女と野獣が戦う2Dアクションゲーム。
キャッチコピーは「歪んだ童話世界を美女と野獣で切り抜け！」
魔法を用いて遠距離攻撃が出来る魔法使いである美女と
剣を振るい魔法を使うリソースを回復させる野獣の2人を
使い分けステージを攻略していく。
製作人数
13 人
製作期間
220 時間
使用ツール
Unity (2019.2.21f1) / Github Desktop
担当箇所
攻撃動作 / スキル管理 / 戦闘中UI管理
リザルト画面作成 / シーン管理
工夫した点
最初のゲーム制作であった為、なるべく同じ処理を書かない
無駄に変数を公開しない、一目見て用途が分かる名称を付ける等
今後のゲーム制作において大事な要素を意識し、実装しました。
苦労した点
キャラクターにLive2Dを使用していたが
描画が崩れたり、不透明度が反映されなかったり
多くのトラブルが相次いだ点。
特にアニメーションが正しく動作しない問題は頭を抱えました。
